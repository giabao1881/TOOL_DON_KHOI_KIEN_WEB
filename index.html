FAQ nhanh
1) Tải về nhưng bị chặn / không tải được?

Nếu file đặt trên GitHub, bạn nên dùng link Release hoặc link trực tiếp (không phải trang xem file). Nếu bạn muốn, mình có thể chỉnh giúp bạn theo chuẩn “Download latest”.

2) Mở file Excel bị cảnh báo Macro?

Đây là cơ chế bảo mật của Excel. Bạn cần bấm Enable Content. Nếu vẫn bị chặn: vào Excel > Options > Trust Center > Trust Center Settings > Macro Settings (tùy chính sách máy).

3) Tôi muốn trang tự cập nhật version/ghi chú?

Bạn chỉ cần chỉnh file dkk-btk-config.json (mục version / changelog / links) rồi push lên GitHub Pages là xong.
